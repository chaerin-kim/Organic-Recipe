## 확인하는 법 

1. root 에서 frontend 폴더로 이동 후 npm start

터미널 작성 : `cd frontend`  > `npm start`

1. root 에서 backend 폴더 이동후 nodemon index.js

터미널 작성 : `cd backend`  > `nodemon index.js`
